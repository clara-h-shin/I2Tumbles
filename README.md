## SimpleTracking

This script was written to utilize TrackingGUI without the GUI and pre-set settings that are frequently used in Baylink Lab. 

- To use this program, open *SimpleTracking.m* from MATLAB.
- You will need to have tiff images converted from a video to run this script properly. To learn more about converting the images, please refer to *swim_tracking_tutorial_by_kailie.docx*.
- Please set the parameters before running.
- For more detailed instructions, refer to *SimpleTracking_Instructions.pdf*.
